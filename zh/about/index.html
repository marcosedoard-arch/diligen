关于 Diligencir — 我们的方法论与团队

在国际大宗商品贸易中，存在一个风险高度集中于数小时之内的时刻：就在你即将电汇定金、签署 SPA、放行装运之前。那一刻，你所依凭的，是通过电子邮件收到的文件、一个从未实地见过的公司名号，以及一个信誓旦旦说货物真实存在的中间人。Diligencir 的存在，正是为了照亮这一时刻。

Diligencir 为巴西与世界之间的大宗商品贸易提供诚信与反欺诈尽职调查——涵盖禽肉、ICUMSA 糖、大豆、玉米、牛肉与咖啡。我们在合同尚未归你之前，就调查清楚坐在桌子另一边的是谁，从而让“推进”或“止步”的决定建立在证据之上，而非盲目的信任。

在交易归你之前，先弄清楚对面是谁。

我们做什么

我们的产品是交易对手 X 光（Counterparty X-Ray）：对企业、其背后的人以及所出示文件进行结构化调查，最终给出一个客观裁决——GO、NO-GO 或 EDD。

  • GO — 我们未发现重大警示信号；在我们所能完成的核查范围内，交易对手与文件经受住了检验。
  • NO-GO — 我们发现了足够严重的迹象，建议你不要按现状推进。
  • EDD（Enhanced Due Diligence，强化尽职调查）— 存在需要在任何决定之前进一步深入或澄清的疑点；信号既非绿灯，也非红灯。

每一项发现都附带各自的置信等级及其证据基础。我们交付的不是一句空泛的意见，而是一份附有证据附录的裁决，清楚说明每一条结论的来源。裁决并非对意图的揣测，而是对核查所揭示之事、以及核查未能支撑之事的一次审慎研判。

交易对手 X 光既服务于需要确认海外买家是否真实存在的巴西出口商，也服务于收到过于激进的报价、想要核实加工厂、贸易商或中间人是否确有其实、是否真有交付能力的外国进口商。

我们的方法论

Diligencir 的权威并非来自墙上的某枚徽章，而是来自方法本身。我们分层开展工作，每一层都留下可审计的痕迹。

1. 文件分析与轻量取证

我们审查所出示的文件（ICPO、SCO、NCNDA、形式发票、各类证书、POP、SBLC 等金融工具），查找内部不一致之处：与所称日期对不上的 PDF 元数据、被反复套用的印章与签名、不连贯的字体与排版、并不存在的注册号。

2. 多法域官方登记核验

我们将交易对手的身份与各相关法域的公开官方来源逐一比对——其中包括巴西的 RFB/CNPJ、智利的 SII、美国的 Sunbiz、巴西律师的 OAB、尼日利亚的 CAC、中国的 GACC，以及巴西用于动物源加工厂与出口商卫生注册的 SIF/MAPA。在我们审查过的一个真实且经匿名化处理的案例中，中间人用作幌子的那家 US LLC，在其声称注册的州的官方登记中根本查无此公司。

3. OSINT 与资产图谱

我们汇集开源情报（OSINT），构建关系与资产图谱：股东、地址、电话、域名、关联企业、过往存在的痕迹。目的在于判断交易对手是否具有实体——还是一个新近注册的空壳。

4. 财务与履约能力分析

我们评估企业是否具备兑现承诺的真实能力：规模与成交量是否相称、是否有经营履历、是否有物流与生产结构的迹象。一个毫无能力支撑却开口就报出数万吨的卖家，本身就是一个信号。

5. 制裁与 PEP 筛查

我们在 KYC、KYC-B 与 AML 框架下，将交易对手及其控制人与制裁名单及 PEP（政治公众人物）暴露情况进行比对筛查。

6. 一手来源确认

只要条件允许，我们就会超越文件本身，在一手来源处加以确认——官方登记本身、签发机关、公开记录。一份证书的价值，只取决于签发它的来源所能证实的内容。

7. 带证据保管链的证据附录

每一项相关要素都以时间戳（timestamp）、来源 URL 与 SHA-256 哈希值加以留存。这便构成一条证据保管链：任何读者在原则上都可以沿原路复核至证据本身，并确认其未被篡改。

8. 逐项发现的置信评级

每一项发现都获得一个置信评级，其依据是来源可靠性 × 信息可信度的组合。这套评级规则支配着裁决的措辞：唯有当一手来源支撑某项发现时，我们才写“已确认”。否则，我们只说“迹象”“提示”或“无法确认”——绝不超出证据所允许的范围。

9. 遵循 LGPD

我们依据LGPD（巴西《通用数据保护法》）处理个人数据，遵循数据最小化原则（仅采集实现目的所必需的数据），并以反欺诈预防的合法依据为基础。我们调查的是诚信——绝不超出目的所需，去窥探任何人的生活。

为什么信任 Diligencir

  • 先求严谨，而非辞令。 任何结论都不会超出证据所能支撑的范围。置信评级规则正是“已确认”与“迹象”之间的分界线。
  • 可复核、有证据支撑的发现。 凭借时间戳、URL 与 SHA-256 哈希值，证据附录让第三方得以沿原路复核——发现并不依赖我们的一面之词。
  • 审慎、表征性的措辞。 我们报告的是迹象，而非道德裁决。这一纪律既保护你、也保护我们，免于因诽谤而承担风险，并使报告在谈判中始终可用。
  • 可触达外国买家无法企及的巴西登记资源。 我们以巴西的语言、时区与行政体系开展工作——RFB/CNPJ、OAB、SIF/MAPA——这是身处亚洲或海湾地区的进口商极难独自核实的。
  • PT/EN 双语报告，并提供 ES/ZH 网站。 交付成果既服务于巴西出口商，也服务于国际交易对手，关键结论不会因翻译而产生歧义。

你将获得什么

交付成果是一份交易对手 X 光报告，其中汇集：

  • 裁决——GO、NO-GO 或 EDD——及其背后的推理依据。
  • 各项发现，每一项均附带其置信等级及由评级规则校准的措辞。
  • 证据附录，逐项附有时间戳、来源 URL 与 SHA-256 哈希值（证据保管链）。
  • 后续步骤建议——在推进之前，应当确认什么、应当要求交易对手提供什么、应当在何处进一步深入。

报告以葡萄牙语和英语呈现，既供决策者阅读，也供审计者查验。

常见问题

你们会指控企业欺诈吗？
不会。Diligencir 既不作出指控，也不作出道德裁决。我们以审慎、表征性的措辞报告迹象——唯有当一手来源支撑某项发现时才写“已确认”；否则只说“迹象”或“无法确认”。这一纪律保护你免于因诽谤而承担风险，并使报告始终可用。
你们以哪些语言、在哪些法域开展工作？
报告以葡萄牙语和英语交付，网站另提供西班牙语与中文。我们通过官方登记在多个法域核验交易对手——其中包括巴西的 RFB/CNPJ、智利的 SII、美国的 Sunbiz、OAB、尼日利亚的 CAC、中国的 GACC 以及巴西的 SIF/MAPA。
你们如何确保一项发现可作为证据？
每一项相关发现都以时间戳、来源 URL 与 SHA-256 哈希值加以留存，构成一条证据保管链。只要条件允许，我们就会在一手来源处加以确认。如此，任何第三方都可沿原路复核至证据本身，并核实其未被篡改。
GO、NO-GO 与 EDD 各是什么意思？
GO 表示我们未发现重大警示信号。NO-GO 表示存在足够严重的迹象，建议不要按现状推进。EDD（强化尽职调查）表示存在需要在任何决定之前进一步深入或澄清的疑点。
